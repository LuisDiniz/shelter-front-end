# Shelter UI

## Pré-requisitos

Node versão 18+. Testado com Node versão 23.11.0

## Iniciar o projeto

1. Navegue até a pasta do projeto:
> cd shelter-front-end

2. Instale as dependências:
> npm install

3. Inicie a applicação:
> npm run dev

4. Acesse o seguinte endereço no navegador:

```
http://localhost:5173/
```
